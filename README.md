# 🐍 Java Swing Snake Game

A classic Snake game built using Java and Swing[cite: 1].<img width="593" height="621" alt="game" src="https://github.com/user-attachments/assets/66c74362-e6cd-497a-8778-567c7b28af78" />
 Navigate the snake to eat apples, grow longer, and score points while avoiding wall and self-collisions!

---

## 🚀 Features

* **Classic Gameplay:** Smooth grid-based movement and collision handling[cite: 1].
* **Score Tracker:** Real-time score display at the top-left corner[cite: 1].
* **Game Over Screen:** Displays final score upon collision[cite: 1].
* **Centered Window:** Automatically centers the game window on launch[cite: 1].

---

## 🛠️ Requirements

* **Java Development Kit (JDK):** Version 8 or higher
* Any Java IDE (e.g., IntelliJ IDEA, Eclipse, VS Code) or command line

---

## 🕹️ Controls

| Key | Action |
| :--- | :--- |
| **`Up Arrow`** | Move Up |
| **`Down Arrow`** | Move Down |
| **`Left Arrow`** | Move Left |
| **`Right Arrow`** | Move Right |
